오늘 처음 올리는 pwnable.kr 문제의 writeup입니다. 이거 말고도 앞으로 2개 더올리겠네요. 그럼 바로 시작하겠습니다.

이번문제는 정말 쉽습니다. 정말정말 쉽습니다. 리눅스의 쉘이 가진 확장이라는 특징을 사용하면 정말쉽게 풀 수 있습니다. 더 자세한 설명은 구글님에게 검색하시면 쉽게 정보를 찾을수 있을 것이라 생각합니다. 


int filter(char* cmd){
	int r=0;
	r += strstr(cmd, "flag")!=0;
	r += strstr(cmd, "sh")!=0;
	r += strstr(cmd, "tmp")!=0;
	return r;
}
여기서 사용자가 입력한 쉘 명령어중 flag, sh, tmp 가 들어가면 안됩니다. 그러나 저희가 열어야하는 파일은 flag라서 flag라는 문제열을 입력하지 않고 flag가 입력한 효과가 나오게 해야합니다. 그렇게 해주는 리눅스 확장 명령어가 *입니다. fla*로 쓰면 flag를 입력한 효과가 나오게 됩니다. 따라서 cat fla*로 입력하면 됩니다.

혹시 틀린부분이나 모르시는 부분이있으시다면 메일로 알려주세요.
